# 專案上手指南

本專案的目的、安裝、運作方式與貢獻說明。

## 快速上手
1. 安裝相依：`npm install` 或 `yarn install`（視專案而定）
2. 建立並啟動本地開發伺服器：`npm run start` / `yarn start`
3. 進行開發與測試：`npm test` / `yarn test`

## 專案結構
- `src/`：前端/後端核心程式碼
- `tests/`：測試
- `docs/`：開發與貢獻說明

## 貢獻指南
- 確認變更影響範圍，提交前跑通測試
- 遵循專案風格與命名慣例
- 提交格式：feat|fix|docs: 說明文字

## 需求與風格
- 依賴管理、測試、CI 等的現況與風格請參考 AGENTS.md
